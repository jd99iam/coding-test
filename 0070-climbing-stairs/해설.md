base case를 topDown 메서드 밖에서 조건을 걸지 말고, 메서드 안에서 if (n==1) , if (n==2) 로 하는 게 더 깔끔하다.

```java
    public static int stair(int n, int[] memo) {
        if (n == 1)
            return 1;
        if (n == 2)
            return 2;
        if (memo[n] == 0) {
            memo[n] = stair(n - 1, memo) + stair(n - 2, memo);
        }
        return memo[n];
    }
```
이렇게 하는게 낫다.
